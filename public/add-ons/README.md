# Add-on card images

Drop real PNG art here, one per add-on, named by its add-on id:

- `achievements.png`
- `shop.png`
- `ai.png`
- `timesheets.png`  (coming soon)
- `marketing.png`   (coming soon)

Recommended ~16:9 (e.g. 800×450). The /add-ons page renders a branded
placeholder automatically for any file that's missing, so partial sets are fine.
